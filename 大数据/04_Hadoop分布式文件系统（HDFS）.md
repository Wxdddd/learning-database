## Hadoop分布式文件系统（HDFS）

### 1. HDFS是什么

​		HDFS（Hadoop Distributed File System）是Hadoop项目的核心子项目，是**分布式计算中数据存储管理的基础**，是基于流数据模式访问和处理超大文件的需求而开发的，可以运行于廉价的商用服务器上。

​		HDFS 源于 Google 在2003年10月份发表的GFS（Google File System） 论文。 它其实就是 GFS 的一个克隆版本。
